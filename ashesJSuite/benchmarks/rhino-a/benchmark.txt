Description: This is JavaScript interpreter written in Java. The
benchmark consists of running a script (in javascript:
testdir/checkParam.js) on a bunch of java source code files found in
testdir/javafiles. Output is to stdout. 

note:
The checkParam.js script is a useful tool  
to check that @param tags in Java documentation comments match the
parameters in the corresponding Java method.
